# Run the Outpost dev server

Start the Express + frontend dev server with:

```bash
bun dev
```

The server runs on port 3000 by default. The frontend dev server runs on port 3001 and is proxied through Express.

After launching, navigate to http://localhost:3000 to verify the app is running.
